Estes são os fontes JAVA e R para construir os graficos dos slides.
Bubble.java executa o bubblesort e escreve na tela o tamanho do array e o tempo que levou para ordenar. Redirecionando a saída para o arquivo sample.txt obtém-se a tabela que é carregada no script plotaula3.R. Este script gera o gráfico de tempo de execução no formato tiff.

Amostraheap.java faz o mesmo para o HeapSort (que estudaremos em aulas futuras). O código do HeapSort está em HeapSort.java

Os scripts funcoes.R e funcoes-aula4.R geram os gráficos para o estudo comparativo de funções (aulas 4 e 5.
